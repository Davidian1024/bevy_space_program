# Bevy Space Program

## Project Goals

A game inspired by [Kerbal Space Program](https://www.kerbalspaceprogram.com) written in [Rust](https://www.rust-lang.org/) using the [Bevy](https://bevyengine.org/) engine.

### Desired Features

- Vehicle Assembly
- Orbital Mechanics
- Aerodynamics
- Thermals
- Interstellar
- Colonies
- Multi-player
